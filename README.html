ğŸ™ï¸ Voiceover Mastery

Equipment (Budget to Pro)

Free/Budget ($0-50) - Your phone's voice recorder - Audacity (free software) - DIY pop filter (sock + hanger)

Mid-Range ($50-200) - USB microphone (Blue Yeti, Audio-Technica ATR2100) - Desktop mic stand - Basic pop filter

Pro Setup ($200-500) - XLR microphone (Shure SM7B, Rode NT1) - Audio interface - Boom arm + shock mount

Recording Tips

  1. Environment - Record in a closet (natural sound dampening)
  2. Distance - 6-8 inches from microphone
  3. Levels - Peak at -12dB, never clip
  4. Script - Print and mark breath points
  5. Warm up - Read aloud for 5 minutes before recording
