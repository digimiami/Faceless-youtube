## Video Upload Checklist

### Before Upload
- [ ] Final edit complete
- [ ] Color correction applied
- [ ] Audio levels normalized
- [ ] Export in 1080p minimum
- [ ] File named properly

### Upload Process
- [ ] Upload to YouTube
- [ ] Add title (SEO optimized)
- [ ] Write description (first 2 lines crucial)
- [ ] Add timestamps (chapters)
- [ ] Add tags (5-8 relevant)
- [ ] Select category
- [ ] Add thumbnail
- [ ] Add end screens
- [ ] Add cards (if relevant)
- [ ] Select language
- [ ] Add subtitles (optional but recommended)
- [ ] Schedule or publish

### After Publishing (First Hour)
- [ ] Watch entire video (check for errors)
- [ ] Reply to first 10 comments
- [ ] Share on Twitter/X
- [ ] Share on relevant communities
- [ ] Send to email list (if applicable)
- [ ] Create YouTube Short version

### After Publishing (First 24 Hours)
- [ ] Reply to all comments
- [ ] Check analytics
- [ ] Pin best comment
- [ ] Share in 3-5 relevant places
- [ ] Engage with similar videos

## SEO Optimization Checklist

### Title Optimization
- [ ] Under 60 characters
- [ ] Main keyword at beginning
- [ ] Power word included
- [ ] Number or year (if relevant)
- [ ] Creates curiosity

### Description Optimization
- [ ] Main keyword in first 25 words
- [ ] 200+ words total
- [ ] Timestamps added
- [ ] Links to related videos
- [ ] Social links included
- [ ] Call-to-action at end

### Tag Strategy
- [ ] 1 main keyword (exact match)
- [ ] 2-3 variations
- [ ] 2-3 broader terms
- [ ] 1-2 specific long-tail
- [ ] No more than 8 tags

### Thumbnail SEO
- [ ] High CTR design
- [ ] Consistent branding
- [ ] Readable text
- [ ] Emotion shown
- [ ] Mobile-friendly

## Emergency Troubleshooting

### Video Performs Poorly

- [ ] Compare to successful videos
- [ ] Adjust and re-test

### Channel Growth Stalls
- [ ] Review content consistency
- [ ] Analyze audience retention

- [ ] Survey audience
- [ ] Try new content types

### Burnout Prevention
