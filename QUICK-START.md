## HOUR 2: Set Up Your Channel

### Create YouTube Channel (20 minutes)
1. Go to youtube.com
2. Click "Sign in" ‚Üí "Create account"
3. Choose "For myself"
4. Fill in name, email, password
5. Click your profile ‚Üí "Create a channel"
6. Name your channel (keep it simple)

### Channel Art (15 minutes)
- **Banner:** Use Canva (1280x720 minimum)
- **Logo:** Canva logo maker or simple text
- **Colors:** Pick 2-3 colors max

### Channel Description (5 minutes)
```
Welcome to [Channel Name]!

We explore [niche topic] with [unique angle].

New videos every [day] at [time].

Subscribe and hit the bell!

For business: [email]
```

## HOUR 4: Record Your Voice

### Free Recording Options
1. **Your phone** - Voice memo app
2. **Audacity** (free) - Download at audacityteam.org
3. **Windows Voice Recorder** - Built-in

### Recording Tips (15 minutes of recording)
1. Find a quiet room
2. Speak 6 inches from mic
3. Read script naturally (don't sound robotic)
4. Pause between sections (for editing)
5. Don't worry about mistakes - we'll edit

### Save your audio file
Name it: `video-001-audio.wav`

## HOUR 6: Edit Your Video

### Free Editing Software
- **CapCut** (Desktop) - capcut.com
- **DaVinci Resolve** - blackmagicdesign.com
- **iMovie** (Mac) - Built-in

### Simple Edit Structure (30 minutes)
1. Import audio track
2. Add background music (20% volume)
3. Add B-roll clips over audio
4. Add text titles for each section
5. Add simple transitions
6. Export in 1080p

### Export Settings
- Format: MP4
- Resolution: 1920x1080 (1080p)
- Frame rate: 30fps
- Audio: 320kbps

## HOUR 8: Upload to YouTube

### Upload Process (20 minutes)
1. Click "Create" ‚Üí "Upload video"
2. Select your video file
3. Add title (under 60 characters)
4. Write description (200+ words)
5. Add 5-8 tags
6. Upload thumbnail
7. Add to playlist (create one)
8. Select language
9. Schedule or publish

### Title Formula
"[Number] [Adjective] [Topic] That [Benefit]"

Example: "10 Surprising Tech Facts That Will Blow Your Mind"

### Description Template
```
In this video, we explore [topic] and reveal [benefit].

Timestamps:
0:00 Introduction
1:30 [Section 1]
3:45 [Section 2]
6:00 [Section 3]
8:30 Conclusion

Resources mentioned:
[Link 1]
[Link 2]

Subscribe for more [niche] content!

#hashtag1 #hashtag2 #hashtag3
```
